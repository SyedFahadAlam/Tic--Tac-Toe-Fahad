Tic Tac Toe App 🎮
A simple, engaging Tic Tac Toe game. 
🚀 Features
Two-Player Mode: Play locally against a friend on the same device.
Turn Tracker: Clearly indicates whose turn it is (X or O).
Win/Tie Detection: Automatically detects a win (horizontally, vertically, or diagonally) or a draw.
Reset Functionality: Easily restart the game after a match ends.
